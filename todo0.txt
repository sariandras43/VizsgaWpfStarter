v1
0. Projekt létrehozása, csomagok telepítése:
	- dotnet add package Microsoft.EntityFrameworkCore -v 8.0.11
	- dotnet add package Microsoft.EntityFrameworkCore.Design -v 8.0.11
	- dotnet add package Microsoft.EntityFrameworkCore.Tools -v 8.0.11
	- dotnet add package MySql.EntityFrameworkCore -v 8.0.11
1. Parancs futtatása buildelés után a Nuget Package Manager Console-ban: Scaffold-dbcontext "Server=localhost;Database=dbneve;Uid=root;Pwd=;" -MySql.EntityFrameworkCore -OutputDir Data
	- Tools > Nuget Package manager > Package Manager Console
((1.1) Ha zavaróak az táblák elnevezései átnevezés az újonnan generált contextben)
2. Database.cs átnevezések: AppContext -> az újonnan generált context (csoportos átnevezés), majd eredeti projektbe átmásolás
3. MainWindow.xaml.cs átnevezések: A szükséges listák átnevezése, megjelenítési útvonal, binding-ok átnevezése XAML-ben
